# Getting started
## Dependencies

- NodeJS

## Installation
1. Truffle
```
npm install -g truffle
```
2. Test Helpers
```
npm install @openzeppelin/test-helpers
```

## Launch tests

1. Launch Ganache in a separate terminal
```
ganache
```

2. Compile and deploy the smart contract
```
truffle migrate
```

3. Run tests
```
truffle test
```
